# Vue Workflow

本项目基于 Vite + Vue 3，集成了 Less、Naive UI、Pinia 和 TailwindCSS。

## 技术栈
- Vue 3
- Vite
- Less
- Naive UI
- Pinia
- TailwindCSS

## 启动项目
```bash
npm run dev
```

## 说明
- 样式入口为 `src/style.less`，已集成 TailwindCSS。
- Naive UI 组件已在 `App.vue` 和 `HelloWorld.vue` 示例中使用。
- Pinia 已在入口注册，可直接创建 store 使用。
